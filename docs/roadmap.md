# 🗺️ Roadmap и Планы Развития: NovaAssist

## Этап 1: Стабилизация Веб-версии (Текущий статус - Почти завершен ✅)
- [x] Разработка базового интерфейса и геймификации.
- [x] Интеграция Google Gemini и системы Fallback (пул из 10 ключей).
- [x] Разделение ИИ на "Базовую Nova" и "Стратега Nova".
- [x] Интеллектуальная премодерация ввода (защита от мусорных названий типа "апапа" и "123123").
- [x] Оптимизация токенов ИИ (отсечение старой истории чата, лимит отображения).
- [x] Страница настроек профиля (Био, Цели, Интересы).
- [ ] Окончательное тестирование и фиксы мелких багов UI перед переходом к БД.

## Этап 2: Облако, Безопасность и Авторизация ☁️
- [ ] **Интеграция БД:** Переход с локального хранилища `localStorage` на Supabase (база данных PostgreSQL).
- [ ] **Токены пользователей:** Хранение баланса токенов в защищенной базе данных на сервере.
- [ ] **Авторизация (Auth):** Вход в приложение через Google и Telegram (в 1 клик для сохранения прогресса).
- [ ] **Интерактивный Онбординг:** Геймифицированное обучение при первой регистрации ("Windows Style" затемнение экрана, ввод профиля, создание первой задачи с Nova).

## Этап 3: Telegram Mini App и Виральность 📱
- [ ] **Интеграция с Telegram (TMA):** Обертка веб-версии в Telegram Web App.
- [ ] **Telegram Auth:** Прозрачная авторизация по ID Telegram без ввода паролей.
- [ ] **Виральные механики:** Реферальная система ("Пригласи друга - получи N токенов ИИ").
- [ ] **Уведомления:** Бот в ТГ присылает Push-напоминания о невыполненных задачах ("Nova скучает по тебе").

## Этап 4: Монетизация (Freemium Модель) 💰
- [ ] **Free Tier:** Возобновляемый бесплатный лимит токенов каждые сутки/неделю (модель `flash`).
- [ ] **Rewarded Ads (Бесплатные токены):** Возможность заработать токены за просмотр короткой рекламы (видео) или подписку на каналы спонсоров.
- [ ] **Nova PRO (Подписка):** Платный тариф (около $3-5/мес или оплата в Telegram Stars).
  - *Безлимитные токены ИИ.*
  - *Глубокая многострочная память и доступ к аналитической модели `gemini-1.5-pro` для Стратега.*
  - *Кастомные градиентные обложки и премиальные темы интерфейса.*
  - *Детальная статистика продуктивности (графики).*

## Этап 5: Нативные Мобильные Приложения 🍏
- [ ] Конвертация на фреймворк React Native (или Expo).
- [ ] Релиз в Apple App Store и Google Play.
- [ ] Интеграция In-App подписок для удобства пользователей вне Telegram.
- [ ] Виджеты на рабочий стол смартфона (быстрый список задач и мотивационная цитата от Nova).
