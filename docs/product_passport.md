# 📂 Паспорт Продукта: NovaAssist

## 1. О Продукте
**NovaAssist** — это геймифицированный ИИ-трекер задач и личный стратег. Продукт помогает пользователям преодолевать прокрастинацию, достигать целей и формировать полезные привычки через систему вознаграждений и глубокое менторство со стороны ИИ (Nova).

## 2. Целевая Аудитория
- Фрилансеры и предприниматели, которым не хватает структуры в течение дня.
- Студенты и люди с СДВГ, нуждающиеся в игровой мотивации по схеме "задача -> награда".
- Все, кто устал от скучных To-Do листов и хочет персонального коуча в кармане.

## 3. Ключевые Фичи (Core Features)
1. **Геймификация:** Выполнение задач приносит "очки", которые можно тратить во встроенном магазине наград "на себя".
2. **Базовый ИИ (Nova):** Быстрый помощник для рутинных операций, умеет добавлять, отмечать и удалять задачи из контекста диалога.
3. **Стратег Nova (Pro-режим):** Ментор для глубокого анализа продуктивности. Помогает планировать расписание (Черновик плана), разгребать завалы задач и рефлексировать. Составляет план действий.
4. **Умная Валидация:** Строгая блокировка бессмысленных задач (случайный набор букв/цифр, например "апапа" или "134343") для поддержания чистоты списка. Подсчет уникальных символов.
5. **Мульти-ключи (Fallback Chain):** Надежная система бесперебойной работы ИИ через серверный пул из 10 API-ключей Google Gemini (до 15 000 текстовых запросов в день бесплатно).
6. **Оптимизация Контекста:** ИИ помнит историю только из последних 6 сообщений, что кардинально снижает расход токенов.

## 4. Технический Стек (Web-версия)
- **Frontend:** React, HTML, Vanilla CSS / Tailwind CSS, Zustand (для state management).
- **ИИ Провайдер:** Google Gemini API (модели `gemini-2.5-flash` для базы и `gemini-1.5-pro` для аналитики).
- **Интеграция ИИ:** Vercel `/api/chat` Serverless Functions (для защиты ключей базы и безопасности).
- **Хранение данных:** Временно `localStorage`, в планах миграция на Supabase (PostgreSQL).

## 5. Уникальное Торговое Предложение (УТП)
Мы не просто список дел. NovaAssist — это единственный инструмент продуктивности, объединяющий **эмоциональную поддержку и коучинг ИИ** с **RPG-механикой наград за жизнь**, делая рутину увлекательной игрой.
